= Kubernetes Terraform Configuration

This Terraform configuration creates a Kubernetes cluster on Oxide using the
Oxide Rancher node driver.

== Requirements

* link:https://developer.hashicorp.com/terraform[`terraform`]

== Usage

. Initialize the Terraform configuration.
+
[source,shell]
----
terraform init
----

. Apply the Terraform configuration.
+
[source,shell]
----
terraform apply
----

. When the apply is finished, take note of the outputs.
+
[source,shell]
----
Apply complete! Resources: 4 added, 0 changed, 0 destroyed.

Outputs:

cluster_name = "oxide-k8s"
cluster_v1_id = "c-m-f7l4lc5h"
kubeconfig = "/Users/example/showcase/integration/rancher/k8s/.kube/config"
rancher_url = "https://45.154.216.222.sslip.io"
----
