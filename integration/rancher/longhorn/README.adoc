= Longhorn Terraform Configuration

This Terraform configuration installs Longhorn on the Kubernetes cluster created
by the link:../k8s[`k8s`] configuration, then runs a Kubernetes job to benchmark
the PVC.

== Usage

. Initialize the Terraform configuration.
+
[source,shell]
----
terraform init
----

. Apply the Terraform configuration.
+
[source,shell]
----
terraform apply
----

. When the apply is finished, the job has successfully completed its `fio` test.
+
[source,shell]
----
Apply complete! Resources: 3 added, 0 changed, 0 destroyed.

Outputs:

fio_job = "fio-test"
longhorn_namespace = "longhorn-system"
longhorn_version = "1.9.1"
----
