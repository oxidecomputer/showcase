= SUSE Linux Micro Packer Configuration

This Packer configuration builds a SUSE Linux Micro image with `open-iscsi`
baked in, so downstream RKE2 nodes can run Longhorn.

== Usage

. Export your Oxide credentials.
+
[source,shell]
----
# Profile.
export OXIDE_PROFILE='my-profile'

# API token.
export OXIDE_HOST='https://oxide.sys.example.com'
export OXIDE_TOKEN='oxide-token-abc123'
----

. Download the `SL-Micro.x86_64-6.2-Base-qcow-GM.qcow2` SUSE Linux Micro image
from https://www.suse.com/download/sle-micro/.

. Convert the image from QCOW2 to raw.
+
[source,shell]
----
qemu-img convert -f qcow2 -O raw SL-Micro.x86_64-6.2-Base-qcow-GM.qcow2 SL-Micro.x86_64-6.2-Base-GM.raw
----

. Upload the raw SUSE Linux Micro image to your Oxide project. See
link:https://docs.oxide.computer/guides/creating-and-sharing-images[Images]
for information on how to do so.

. Copy the example variables file and replace the values with your own.
+
[source,shell]
----
cp secrets.pkrvars.hcl.example secrets.pkrvars.hcl
----
+
.secrets.pkrvars.hcl
[source,hcl]
----
project = "my-project"

# Source image settings.
source_image_name = "suse-linux-micro-6-2-base-gm"
# Uncomment if using a project image instead of a silo image.
# source_image_project = "my-project"

# SUSE Customer Center (SCC) information.
scc_regcode = "ADFEXXXXXXXXXXXX"
scc_email   = "you@example.com"

# Optional: Configure specific network settings.
# ip_pool = "my-ip-pool"
# vpc     = "default"
# subnet  = "default"
----

. Initialize the Packer configuration.
+
[source,shell]
----
packer init .
----

. Build the image.
+
[source,shell]
----
packer build -var-file=secrets.pkrvars.hcl .
----

. When the build is finished, the resulting image name and ID will be shown.
Take note of this image ID as it'll be used in subsequent sections.
+
[source,shell]
----

----
