= Node Driver Terraform Configuration

This Terraform configuration configures the Oxide Rancher Node Driver on Rancher.

== Usage

. Create your `terraform.tfvars` file.
+
.terraform.tfvars
[source,hcl]
----
# The node driver will use the credentials from this Oxide profile.
oxide_profile = "my-profile"
----

. Initialize the Terraform configuration.
+
[source,shell]
----
terraform init
----

. Apply the Terraform configuration.
+
[source,shell]
----
terraform apply
----

. When the apply is finished, take note of the outputs.
+
[source,shell]
----
Apply complete! Resources: 2 added, 0 changed, 0 destroyed.

Outputs:

cloud_credential_id = "cattle-global-data:cc-oxide"
----
